# Financial Risk & Credit Scoring Analytics
Market Risk (VaR) and Credit Risk (Machine Learning) analysis using Python.

## Overview
This repository contains two core financial risk modules:
1. **Market Risk:** Calculating Value at Risk (VaR) for ATX-listed equities.
2. **Credit Risk:** A Machine Learning model predicting loan defaults.

## Technologies
* Python (Pandas, NumPy, Scikit-Learn)
* Data Visualization (Matplotlib, Seaborn)

## Key Results
* **Accuracy:** Achieved 81% accuracy in predicting credit defaults.
* **Risk Insights:** Identified 'Loan Amount' and 'Applicant Age' as the top risk drivers.
* **VaR Analysis:** Quantified 95% confidence interval losses for Austrian portfolios.
